$ cat /etc/clawcondos/description.txt
A goals-first multi-agent dashboard with an embedded apps platform.
Organize AI agents into condos. Embed any app with an AI assistant.
No build step. No framework. Vanilla HTML/JS/CSS. Edit and refresh.
$ clawcondos --list-features
[1]

GOALS_ORGANIZATION

Group agent sessions into high-level goals ("Condos"). Each condo has its own context, history, and embedded tools.

[2]

APPS_PLATFORM

Register any web application. Get it embedded with an AI assistant sidebar that knows your code, logs, and stack.

[3]

REALTIME_STREAMING

WebSocket-based responses with tool activity indicators. Watch agents think, plan, and execute live.

[4]

ZERO_BUILD_STEP

No webpack. No bundlers. No transpilers. Edit HTML/CSS/JS files and refresh the browser.
